# Onepager Template

Eine einfache, moderne Onepager-Website als „Visitenkarte" mit Kontaktformular.

## Live Demo

**https://obm-onepager-template.netlify.app**

## Features

- ✅ Hero-Bereich mit Call-to-Action
- ✅ Über mich / Unternehmen Section
- ✅ Leistungen (4 Boxen)
- ✅ Kontaktbereich mit Formular
- ✅ Impressum & Datenschutz
- ✅ Responsive Design (Mobile First)
- ✅ Netlify Forms Integration
- ✅ Smooth Scroll Navigation

## Technik

- **HTML5** - Semantischer Aufbau
- **CSS3** - Modernes Layout mit CSS Grid & Flexbox
- **Vanilla JS** - Keine Abhängigkeiten
- **Netlify** - Hosting & Form Handling

## Struktur

```
onepager-template/
├── index.html          # Hauptseite (Onepager)
├── impressum.html      # Impressum
├── datenschutz.html    # Datenschutzerklärung
├── css/
│   └── style.css      # Styles
├── js/
│   └── main.js        # JavaScript
└── netlify.toml       # Netlify Config
```

## Anpassung

Alle Platzhalter sind mit `[...]` markiert:

- `[Firmenname]` - Name des Unternehmens
- `[Hier steht...]` - Text-Platzhalter
- `[LOGO]` - Logo einfügen
- `[Adresse]` - Kontaktdaten

## Deployment

1. Repo forken/klonen
2. Inhalte anpassen
3. Auf Netlify verbinden
4. Fertig!

## Kontaktformular

Das Formular nutzt **Netlify Forms**:
- Felder: Name, E-Mail, Nachricht
- DSGVO-Checkbox
- Automatische E-Mail-Benachrichtigung
